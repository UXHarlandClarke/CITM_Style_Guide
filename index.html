Introduction

The CITM UI Toolkit is built from a customized SASS port of Bootstrap. It utilizes Bootstrap’s responsive grid system, which is demostrated below. Visit Bootstrap’s site for a more detailed explanation of grid options.

When hovering over any component in the toolkit, a < > button will appear. Clicking this button opens a modal to the source code for that component for easy copying.
